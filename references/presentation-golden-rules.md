# Presentation Golden Rules

Use this reference when creating production slide briefs, reviewing an existing deck, or explaining the principles behind a deck plan.

## Classic Method Basis

- **Pyramid Principle**: Put the top-level answer first, then organize supporting arguments and evidence underneath it. Use this for executive and decision-maker briefings.
- **Duarte narrative contrast**: Move the audience between "what is" and "what could be" so the deck has tension, relevance, and a reason to act.
- **Assertion-Evidence**: Write each slide around a complete assertion and support it with visual evidence, rather than writing a topic heading above bullet lists.
- **Presentation Zen**: Prefer restraint, simplicity, and naturalness. Remove visual and textual clutter that competes with the message.
- **Tufte information design**: Preserve evidence quality, context, and integrity. Avoid chart junk, misleading axes, decorative effects, and graphics that hide the data.
- **Audience empathy**: Design the deck around the audience's role, beliefs, objections, decision rights, and cost of change.

## Page Self-Check

Before accepting a slide or slide brief, check:

- Is the title a judgment or conclusion rather than a topic label?
- Does the page make only one core point?
- Does the body support the title directly?
- Can the audience identify the point within five seconds?
- Does the page answer "So what?" for the target audience?
- Can anything be deleted without hurting comprehension?
- Do charts include the needed scope, unit, time range, source, and comparison basis?
- Does the page move the audience toward a decision or next action?

## Planning Checks

- Start from the desired audience action, then work backward to the narrative.
- Build a message map before the page list.
- Keep the main deck focused on decision flow; move verification depth into appendix.
- Reframe the same content by audience role instead of reusing a generic deck.
- Treat speaker notes as delivery support, not as visible slide content.

## Source Anchors

- Duarte: audience empathy and "what is" to "what could be" contrast.
- Minto / Pyramid Principle: conclusion-first hierarchical communication.
- Michael Alley / Assertion-Evidence: assertion headline plus visual evidence.
- Garr Reynolds / Presentation Zen: restraint, simplicity, naturalness.
- Edward Tufte: information integrity and chart clarity.
